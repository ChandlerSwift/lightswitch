Theory

Note: This documentation is mostly complete, but the web control page should serve as the reference implementation.

Each switchable output is represented as a light. The built-in LED is numbered 0, the main relay-driven light is 1, the red channel is 2, and so on. Each light can be turned on and off, and its status retrieved.

Light

For each command, ID should be replaced with the light's integer value (0-5).

  • Turn on the light: /light/on?id=ID
  • Turn off the light: /light/off?id=ID
  • Get light status: /light/status?id=ID
